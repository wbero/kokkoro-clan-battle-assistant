# 时钟数字模板

## 模板加载优先级

程序按以下顺序查找模板，找到即用：

1. `/sdcard/Download/可可萝自动会战助手/templates/` — **用户自定义（最高优先）**
2. `assets/templates/` (APK 内置) — 打包在 APK 里的默认模板

你在工作目录放入同名文件即可覆盖 APK 内置模板。

## 手动裁剪步骤

1. 在真机上进入战斗画面，截一张图
2. 把截图传到电脑上
3. 找到计时器位置（右上角，格式 `M:SS`，如 `1:16`）
4. 逐一裁剪以下字符，保存为 PNG：

```
0.png    1.png    2.png    3.png    4.png
5.png    6.png    7.png    8.png    9.png
colon.png  （冒号 :）
```

### 裁剪要求

- **紧贴字符左右边缘**，不留白边
- **所有模板高度必须一致**（非常重要！）
  - 数字多高，冒号裁剪框就多高
  - 冒号本体矮，但裁剪框要和数字同高，冒号居中，上下留白
  - 高度不一致会被测试脚本检测出来
- 宽度可以不一致（1 比 0 窄是正常的）
- PNG 格式

```
正确示例（数字和冒号都是 18px 高）：
┌────┐  ┌──┐  ┌────┐  ┌────┐
│    │  │  │  │    │  │    │
│ 1  │  │: │  │ 0  │  │ 8  │
│    │  │  │  │    │  │    │
└────┘  └──┘  └────┘  └────┘
 8×18   6×18  12×18  12×18
```

### 技巧

截一张包含 `1:08` 或 `1:18` 的图（数字种类多），
一张图就能裁出 0、1、8 和冒号。其他数字分几次补全。

## 部署

放入工作目录：
```
/sdcard/Download/可可萝自动会战助手/templates/
  0.png ... 9.png  colon.png
```

或放在 `assets/templates/` 目录随 APK 打包。
