# 坐标与时间区域校准

配置文件名为 `可可萝自动会战助手.txt`，放在：

```text
/sdcard/Download/可可萝自动会战助手/
```

首版配置文件共 14 行：

```text
第1行：5号位角色头像 X 坐标
第2行：4号位角色头像 X 坐标
第3行：3号位角色头像 X 坐标
第4行：2号位角色头像 X 坐标
第5行：1号位角色头像 X 坐标
第6行：所有角色头像 Y 坐标
第7行：AUTO 按钮 X 坐标
第8行：AUTO 按钮 Y 坐标
第9行：黑屏/起点监测 X 坐标，首版保留
第10行：黑屏/起点监测 Y 坐标，首版保留
第11行：时间区域左上 X 坐标
第12行：时间区域左上 Y 坐标
第13行：时间区域宽度
第14行：时间区域高度
```
